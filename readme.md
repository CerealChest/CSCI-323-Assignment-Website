# Cole D. Miles — Academic Site (Astro + TypeScript + Tailwind + MDX)

A lightweight, static personal academic website template (Astro + TypeScript + Tailwind + MDX) built for direct deployment to **GitHub Pages**. No backend required — content is local (JSON/MD/MDX).

## Features
- Static-only (no server)
- Astro + TypeScript + MDX
- Tailwind CSS, responsive layout
- Dark / Light theme-aware
- Sample content & profile data in `/data`
- Deploy script using `gh-pages` to publish `dist/` to `gh-pages` branch
